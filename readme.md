# Color Pick Wheel

## Functionality

* Multiple Color Set
* Brightness Slider


## How to Run

To run this project, you have to serve html files.

Please follow commands below.

1. Install node_modules

   ```
   npm install
   ```
2. Serve in dev mode

   ```
   npm run dev
   ```
3. Play Color-Pick-Wheel

   Open [http://localhost:5173/](http://localhost:5173/) in your Web Browser.
